# Whisperಕೊಲಾಬ್

Whisperಪ್ರತಿಲೇಖನಕ್ಕಾಗಿ ಬಹುಭಾಷಾ Google Colabನೋಟ್‌ಬುಕ್‌ಗಳು.

## ವೈಶಿಷ್ಟ್ಯಗಳು

- **ಬಳಸಲು ಉಚಿತ:** ಈ ನೋಟ್‌ಬುಕ್ [openai/whisper](https://github.com/openai/whisper)ಅನ್ನು ಬಳಸುತ್ತದೆ, ಇದು OpenAIಜನವರಿ 17, 2023 ರಂದು ಮುಕ್ತ-ಮೂಲವನ್ನು Google Colabನ ಉಚಿತ GPUರನ್‌ಟೈಮ್ ಜೊತೆಗೆ.
- **ಯಾವುದೇ ಸ್ಥಿರ ನೋಟ್‌ಬುಕ್ ಬದಿಯ ಫೈಲ್ ಗಾತ್ರದ ಮಿತಿಯಿಲ್ಲ:** ದೊಡ್ಡ ಆಡಿಯೋ ಮತ್ತು ವಿಡಿಯೋ ಫೈಲ್‌ಗಳನ್ನು ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಬಹುದು.
- **ಬ್ಯಾಚ್ ಸಂಸ್ಕರಣೆ:** ಬಹು ಆಡಿಯೋ ಮತ್ತು ವಿಡಿಯೋ ಫೈಲ್‌ಗಳನ್ನು ಒಂದು ರನ್‌ನಲ್ಲಿ ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಬಹುದು.
- **Google Driveಬೆಂಬಲ:** ಫೈಲ್‌ಗಳನ್ನು ನೇರವಾಗಿ ಅಪ್‌ಲೋಡ್ ಮಾಡಬಹುದು ಅಥವಾ Google Driveನಲ್ಲಿ ಸಂಗ್ರಹಿಸಲಾದ ಆಡಿಯೊ/ವೀಡಿಯೊ ಫೈಲ್‌ಗಳನ್ನು ಪ್ರಕ್ರಿಯೆಗೊಳಿಸಬಹುದು.
- **ಸಂಪೂರ್ಣವಾಗಿ ಆನ್‌ಲೈನ್:** ಡೌನ್‌ಲೋಡ್‌ಗಳು ಮತ್ತು ಪ್ರಕ್ರಿಯೆಯನ್ನು Google Colabನಲ್ಲಿ ನಿರ್ವಹಿಸಲಾಗುತ್ತದೆ. ಯಾವುದೇ ಸ್ಥಳೀಯ ಕಂಪ್ಯೂಟಿಂಗ್ ಸಂಪನ್ಮೂಲಗಳನ್ನು ಬಳಸಲಾಗುವುದಿಲ್ಲ.
- **ಬಹು ಔಟ್ಪುಟ್ ಸ್ವರೂಪಗಳು:** ನೋಟ್‌ಬುಕ್ ಮೂಲ JSONಪ್ರತಿಲೇಖನವನ್ನು ಔಟ್‌ಪುಟ್ ಮಾಡುತ್ತದೆ ಮತ್ತು TXT, CSV, ಮಾರ್ಕ್‌ಡೌನ್, Excel, ಮತ್ತು SRTಉಪಶೀರ್ಷಿಕೆ ಫೈಲ್‌ಗಳನ್ನು ಪಡೆಯಲಾಗಿದೆ.
