# Whisperكولاب

دفاتر ملاحظات Google Colabمتعددة اللغات للنسخ Whisper.

## الميزات

- **مجاني للاستخدام:** This notebook uses [openai/whisper](https://github.com/openai/whisper), which OpenAIopen-sourced on January 17, 2023, together with Google Colab's free GPUruntime.
- **لا يوجد حد ثابت لحجم الملف على جانب دفتر الملاحظات:** يمكن معالجة ملفات الصوت والفيديو الكبيرة.
- **تجهيز الدفعة:** يمكن معالجة ملفات الصوت والفيديو المتعددة في تشغيل واحد.
- **Google Driveالدعم:** Files can be uploaded directly, or audio/video files stored in Google Drivecan be processed.
- **Fully online:** Downloads and processing are performed on Google Colab. لا يتم استخدام أي موارد حوسبة محلية.
- **تنسيقات إخراج متعددة:** The notebook outputs the original JSONtranscript and derived TXT, CSV, Markdown, Excel, and SRTsubtitle files.

## تخطيط المستودع

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

الجذر READMEهو نقطة الإدخال باللغة الإنجليزية، ويحتوي docs/READMEعلى ملفات READMEمترجمة تم إنشاؤها. دفتر ملاحظات القالب هو دفتر الملاحظات المصدر ويحتوي على عناصر نائبة مثل `{{cell1.title}}`. توفر ملفات الإعدادات المحلية READMEونسخة مواجهة لـ Colab خاصة باللغة. يتم الالتزام بدفاتر الملاحظات التي تم إنشاؤها بحيث تظل روابط Colab مستقرة.

## بناء

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## الشيكات

```bash
python scripts/check_notebooks.py
```

يقوم سير عمل CI بإعادة بناء الملفات والدفاتر READMEمن القالب وملفات الإعدادات المحلية، ويفحص دفتر الملاحظات JSON، ويتحقق من المخرجات الفارغة، ويتحقق من بناء جملة Python في خلايا التعليمات البرمجية، ويبحث عن الأسرار العرضية الواضحة، ويفشل عندما لا يتم الالتزام بالملفات التي تم إنشاؤها.

## ما الذي يمكن لـ CI اختباره

- إنشاء READMEمن الملفات المحلية
- إنشاء دفتر ملاحظات من القالب وملفات اللغة
- دفتر الملاحظات JSONالصلاحية
- مخرجات التنفيذ الفارغة
- بناء جملة بايثون في خلايا التعليمات البرمجية
- أسرار عرضية واضحة
- تم إنشاء READMEوملفات دفتر الملاحظات المطابقة للمصادر

يتطلب مسار النسخ الكامل واجهات برمجة التطبيقات الخاصة بـ Colab، وتنزيلات النماذج، والتوفر GPU، وملفات الوسائط المقدمة من المستخدم. تعامل مع هذا المسار كاختبار يدوي لدخان Colab بعد اجتياز عمليات فحص الملفات التي تم إنشاؤها.

## الترخيص

تم إصدار المولد READMEلهذا المستودع، وقالب دفتر الملاحظات، والملفات المحلية، ودفاتر الملاحظات التي تم إنشاؤها، والبرامج النصية للصيانة بموجب ترخيص MIT. تخضع نماذج Whisperوتبعيات الجهات الخارجية للتراخيص الخاصة بها.
