# WhisperColab

Viacjazyčné zápisníky Google Colabna prepis Whisper.

## Vlastnosti

- **Bezplatné použitie:** Tento notebook používa [openai/whisper](https://github.com/openai/whisper), ktorý OpenAIsprístupnil 17. januára 2023, spolu s bezplatným runtime GPUod Google Colab.
- **Žiadny pevný limit veľkosti súboru na strane notebooku:** Je možné spracovať veľké audio a video súbory.
- **Dávkové spracovanie:** V jednom spustení je možné spracovať viacero audio a video súborov.
- **Podpora Google Drive:** Súbory je možné nahrať priamo alebo spracovať audio/video súbory uložené v Google Drive.
- **Plne online:** Sťahovanie a spracovanie sa vykonáva na Google Colab. Nepoužívajú sa žiadne lokálne výpočtové zdroje.
- **Viac výstupných formátov:** Výstupom notebooku je pôvodný prepis JSONa odvodené súbory TXT, CSV, Markdown, Excela SRT.

## Rozloženie úložiska

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

Koreň READMEje anglický vstupný bod a docs/READMEobsahuje vygenerované lokalizované súbory README. Šablóna poznámkového bloku je zdrojový poznámkový blok a obsahuje zástupné symboly, ako napríklad `{{cell1.title}}`. Súbory miestnych nastavení poskytujú pre jazyk READMEa kópiu na strane Colab. Vygenerované zápisníky sú zaviazané, takže prepojenia Colab zostanú stabilné.

## Stavať

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Kontroly

```bash
python scripts/check_notebooks.py
```

Pracovný postup CI prebuduje súbory READMEa poznámkové bloky zo súborov šablón a miestnych nastavení, skontroluje poznámkový blok JSON, overí prázdne výstupy, skontroluje syntax Pythonu v bunkách kódu, vyhľadá zjavné náhodné tajomstvá a zlyhá, keď vygenerované súbory nie sú potvrdené.

## Čo môže CI testovať

- Generovanie READMEzo súborov miestnych nastavení
- generovanie poznámkového bloku zo súborov šablón a miestnych nastavení
- zápisník JSONlatnosť
- prázdne výstupy vykonávania
- Syntax Pythonu v bunkách kódu
- zjavné náhodné tajomstvá
- commit vygeneroval READMEa súbory poznámkového bloku zodpovedajúce zdrojom

Úplná cesta prepisu vyžaduje rozhrania API špecifické pre Colab, sťahovanie modelov, dostupnosť GPUa mediálne súbory poskytnuté používateľom. Po úspešnom vykonaní kontrol vygenerovaných súborov túto cestu berte ako manuálny test dymu Colab.
