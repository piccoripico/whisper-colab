# WhisperColab

Multilingual Google Colabna mga notebook para sa Whispertranskripsyon.

## Mga tampok

- **Libreng gamitin:** Ang notebook na ito ay gumagamit ng [openai/whisper](https://github.com/openai/whisper), na OpenAIopen-sourced noong Enero 17, 2023, kasama ng Google Colabna libreng GPUruntime.
- **Walang nakapirming limitasyon sa laki ng file sa gilid ng notebook:** Maaaring iproseso ang malalaking audio at video file.
- **Batch processing:** Maaaring iproseso ang maraming audio at video file sa isang pagtakbo.
- **Google Drivesuporta:** Maaaring direktang i-upload ang mga file, o maaaring iproseso ang mga file na audio/video na nakaimbak sa Google Drive.
- **Ganap na online:** Ang mga pag-download at pagproseso ay isinasagawa sa Google Colab. Walang ginagamit na lokal na mapagkukunan ng computing.
- **Maramihang mga format ng output:** Ang notebook ay naglalabas ng orihinal na JSONtranscript at nagmula sa TXT, CSV, Markdown, Excel, at SRTna mga subtitle na file.

## Layout ng Imbakan

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

Ang ugat na READMEay ang English entry point, at ang docs/READMEay naglalaman ng mga nabuong READMEna file. Ang template na notebook ay ang source notebook at naglalaman ng mga placeholder gaya ng `{{cell1.title}}`. Ang mga lokal na file ay nagbibigay ng tukoy sa wika na READMEat nakaharap sa Colab na kopya. Naka-commit ang mga nabuong notebook para manatiling stable ang mga link ng Colab.

## Bumuo

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Mga tseke

```bash
python scripts/check_notebooks.py
```

Ang daloy ng trabaho ng CI ay muling nagtatayo ng READMEna mga file at notebook mula sa template at mga lokal na file, sinusuri ang notebook JSON, bini-verify ang mga walang laman na output, sinusuri ang Python syntax sa mga cell ng code, nag-scan para sa mga malinaw na hindi sinasadyang lihim, at nabigo kapag ang mga nabuong file ay hindi ginawa.

## Ano ang Masusubok ng CI

- READMEhenerasyon mula sa mga lokal na file
- pagbuo ng notebook mula sa template at mga lokal na file
- notebook JSONvalidity
- walang laman na mga output ng pagpapatupad
- Syntax ng Python sa mga cell ng code
- malinaw na hindi sinasadyang mga lihim
- nakagawa ng READMEat mga notebook na file na tumutugma sa mga source

Ang buong transcription path ay nangangailangan ng mga Colab-specific na API, pag-download ng modelo, GPUavailability, at media file na ibinigay ng user. Tratuhin ang landas na iyon bilang isang manu-manong pagsubok sa usok ng Colab pagkatapos pumasa sa mga nabuong-file na pagsusuri.

## Lisensya

Ang READMEgenerator, notebook template, locale file, nabuong notebook, at maintenance script ng repository na ito ay inilabas sa ilalim ng MITLicense. Ang Whispermga modelo at mga dependency ng third-party ay pinamamahalaan ng kani-kanilang mga lisensya.
