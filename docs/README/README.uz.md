# WhisperKolab

Whispertranskripsiyasi uchun ko'p tilli Google Colabnoutbuklari.

## Xususiyatlari

- **Foydalanish bepul:** This notebook uses [openai/whisper](https://github.com/openai/whisper), which OpenAI open-sourced on January 17, 2023, together with Google Colab's free GPU runtime.
- **Noutbuk tomonidagi fayl oʻlchamiga aniq cheklov yoʻq:** Katta hajmdagi audio va video fayllarni qayta ishlash mumkin.
- **To'plamni qayta ishlash:** Bir ishga tushirishda bir nechta audio va video fayllarni qayta ishlash mumkin.
- **Google Driveqo'llab-quvvatlash:** Fayllarni toʻgʻridan-toʻgʻri yuklash yoki Google Driveda saqlangan audio/video fayllarni qayta ishlash mumkin.
- **To'liq onlayn:** Yuklab olish va qayta ishlash Google Colaborqali amalga oshiriladi. Mahalliy hisoblash resurslaridan foydalanilmaydi.
- **Bir nechta chiqish formatlari:** Notebook asl JSONtranskriptini va olingan TXT, CSV, Markdown, Excelva SRTsubtitr fayllarini chiqaradi.

## Repozitoriy tartibi

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

READMEildizi inglizcha kirish nuqtasi boʻlib, docs/READMEmahalliylashtirilgan READMEfayllarini oʻz ichiga oladi. Daftar daftar manba daftar bo‘lib, `{{cell1.title}}`kabi to‘ldiruvchilarni o‘z ichiga oladi. Mahalliy fayllar tilga xos READMEva Colab-ga tegishli nusxani taqdim etadi. Yaratilgan noutbuklar Colab havolalari barqaror bo'lib qolishi uchun sodiqdir.

## Qurish

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Tekshiruvlar

```bash
python scripts/check_notebooks.py
```

CI ish jarayoni READMEfayllari va daftarlarini shablon va mahalliy fayllardan qayta tiklaydi, JSONdaftarini tekshiradi, boʻsh chiqishlarni tekshiradi, kod hujayralarida Python sintaksisini tekshiradi, aniq tasodifiy sirlarni skanerlaydi va yaratilgan fayllar bajarilmaganda muvaffaqiyatsiz boʻladi.

## CI nimani sinab ko'rishi mumkin

- Mahalliy fayllardan READMEyaratish
- shablon va mahalliy fayllardan notebook yaratish
- daftar JSONamal qilish muddati
- bo'sh ijro natijalari
- Kod hujayralarida Python sintaksisi
- aniq tasodifiy sirlar
- yaratilgan READMEva manbalarga mos keladigan notebook fayllari

Toʻliq transkripsiya yoʻli uchun Colab-ga xos API’lar, yuklab olish modellari, GPUmavjudligi va foydalanuvchi tomonidan taqdim etilgan media fayllar kerak boʻladi. Yaratilgan fayl tekshiruvlaridan o'tgandan so'ng, bu yo'lni qo'lda Colab tutun sinovi sifatida ko'ring.

## Litsenziya

Ushbu omborning READMEgeneratori, daftar shabloni, mahalliy fayllar, yaratilgan noutbuklar va texnik xizmat koʻrsatish skriptlari MITlitsenziyasi ostida chiqariladi. Whispermodellari va uchinchi tomon bog'liqliklari tegishli litsenziyalari bilan boshqariladi.
