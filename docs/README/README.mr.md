# Whisperकोलाब

Whisperलिप्यंतरणासाठी बहुभाषिक Google Colabनोटबुक.

## वैशिष्ट्ये

- **वापरण्यासाठी मोफत:** हे नोटबुक [openai/whisper](https://github.com/openai/whisper)वापरते, जे OpenAI17 जानेवारी, 2023 रोजी मुक्त-स्रोत, Google Colabच्या विनामूल्य GPUरनटाइमसह.
- **कोणतीही निश्चित नोटबुक-साइड फाइल आकार मर्यादा नाही:** मोठ्या ऑडिओ आणि व्हिडिओ फाइल्सवर प्रक्रिया केली जाऊ शकते.
- **बॅच प्रक्रिया:** एका रनमध्ये अनेक ऑडिओ आणि व्हिडिओ फाइल्सवर प्रक्रिया केली जाऊ शकते.
- **Google Driveसमर्थन:** फाइल्स थेट अपलोड केल्या जाऊ शकतात किंवा Google Driveमध्ये संग्रहित केलेल्या ऑडिओ/व्हिडिओ फाइल्सवर प्रक्रिया केली जाऊ शकते.
- **पूर्णपणे ऑनलाइन:** डाउनलोड आणि प्रक्रिया Google Colabवर केली जाते. कोणतीही स्थानिक संगणकीय संसाधने वापरली जात नाहीत.
- **एकाधिक आउटपुट स्वरूप:** नोटबुक मूळ JSONउतारा आणि TXT, CSV, मार्कडाउन, Excel, आणि SRTउपशीर्षक फाइल्स आउटपुट करते.

## रेपॉजिटरी लेआउट

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

रूट READMEहा इंग्रजी एंट्री पॉइंट आहे, आणि docs/READMEमध्ये जनरेट केलेल्या READMEफायली असतात. टेम्पलेट नोटबुक हे स्त्रोत नोटबुक आहे आणि त्यात `{{cell1.title}}`सारखे प्लेसहोल्डर आहेत. लोकेल फाइल भाषा-विशिष्ट READMEआणि कोलाब-फेसिंग कॉपी प्रदान करतात. व्युत्पन्न केलेल्या नोटबुक वचनबद्ध आहेत त्यामुळे Colab लिंक स्थिर राहतील.

## बांधा

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## चेक करतो

```bash
python scripts/check_notebooks.py
```

CI वर्कफ्लो READMEफाईल्स आणि नोटबुक टेम्पलेट आणि लोकेल फाइल्समधून पुन्हा बनवते, नोटबुक JSONतपासते, रिकाम्या आउटपुटची पडताळणी करते, कोड सेलमध्ये पायथन सिंटॅक्स तपासते, स्पष्ट आकस्मिक गुपितांसाठी स्कॅन करते आणि व्युत्पन्न केलेल्या फायली बांधल्या जात नाहीत तेव्हा अयशस्वी होतात.

## सीआय काय चाचणी करू शकते

- READMEलोकॅल फाइल्समधून जनरेशन
- टेम्पलेट आणि लोकेल फाइल्समधून नोटबुक निर्मिती
- नोटबुक JSONवैधता
- रिक्त अंमलबजावणी आउटपुट
- कोड सेलमध्ये पायथन सिंटॅक्स
- स्पष्ट अपघाती रहस्ये
- वचनबद्ध व्युत्पन्न READMEआणि नोटबुक फायली स्त्रोतांशी जुळतात

पूर्ण ट्रान्सक्रिप्शन पाथसाठी Colab-विशिष्ट API, मॉडेल डाउनलोड, GPUउपलब्धता आणि वापरकर्त्याने पुरवलेल्या मीडिया फाइल आवश्यक आहेत. व्युत्पन्न-फाइल चेक उत्तीर्ण झाल्यानंतर मॅन्युअल Colab स्मोक चाचणी म्हणून त्या मार्गाचा वापर करा.

## परवाना

या भांडाराचे READMEजनरेटर, नोटबुक टेम्पलेट, लोकॅल फाइल्स, व्युत्पन्न केलेल्या नोटबुक्स आणि देखभाल स्क्रिप्ट MITपरवान्याअंतर्गत प्रसिद्ध केल्या जातात. Whisperमॉडेल्स आणि तृतीय-पक्ष अवलंबित्व त्यांच्या संबंधित परवान्यांद्वारे नियंत्रित केले जातात.
