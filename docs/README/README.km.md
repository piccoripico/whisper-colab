- **ឥតគិតថ្លៃក្នុងការប្រើប្រាស់:** សៀវភៅកត់ត្រានេះប្រើ [openai/whisper](https://github.com/openai/whisper)ដែល OpenAIប្រភពបើកចំហនៅថ្ងៃទី 17 ខែមករា ឆ្នាំ 2023 រួមជាមួយនឹង Google Colabពេលដំណើរការ GPUឥតគិតថ្លៃ។
