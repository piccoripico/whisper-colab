## Ezali ndenge nini? Bobongisi ya ebombelo

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

Ezali ndenge nini? Mosuni READMEezali esika ya bokoti ya Lingelesi, mpe docs/READMEezali na ba fichiers localisés READMEoyo esalemi. Kaye ya modèle ezali kaye ya source mpe ezali na ba placeholders lokola `{{cell1.title}}`. Ba fisyé ya esika epesaka READMEoyo etali monoko mpe kopi oyo etali Colab. Ba cahiers générés e committé donc ba liens ya Colab ezala stable.

## Ezali ndenge nini? Tongela

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Ezali ndenge nini? Ba chèques

```bash
python scripts/check_notebooks.py
```

Ezali ndenge nini? Flux ya mosala ya CI etongaka lisusu ba fichiers READMEmpe ba cahiers uta na ba fichiers ya modèle mpe ya locale, e vérifier cahier JSON, e vérifier ba sorties ya pamba, e vérifier syntaxe ya Python na ba cellules ya code, e scanner ba secrets ya accident ya polele, mpe elongi te tango ba fichiers oyo esalemi esalemi te.

- Ezali ndenge nini? READMEgénération uta na ba fichiers ya locale
- Ezali ndenge nini? génération ya cahier à partir ya ba fichiers ya modèle na locale
- Ezali ndenge nini? kaye JSONbosolo
- Ezali ndenge nini? ba sorties ya exécution ya pamba
- Ezali ndenge nini? Syntaxe ya python na ba cellules ya code
- Ezali ndenge nini? basekele ya polele oyo esalemaki na mbalakaka
- Ezali ndenge nini? committed ebimisaki READMEmpe ba fichiers ya kaye oyo ekokani na ba sources

Ezali ndenge nini? Nzela ya transcription mobimba esengaka ba API spécifiques ya Colab, ba téléchargements ya modèle, disponibilité ya GPU, mpe ba fichiers ya media oyo mosaleli apesi. Traite nzela wana lokola test ya milinga ya Colab manuel sima ya ba vérifications ya fichier generated eleki.
