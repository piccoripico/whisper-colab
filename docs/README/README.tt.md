# WhisperКолаб

Күп телләрдә Google Colabдәфтәрләр Whisperтранскрипциясе өчен.

## Featuresзенчәлекләр

- **Куллану өчен бушлай:** Бу дәфтәрдә [openai/whisper](https://github.com/openai/whisper)кулланыла, бу OpenAI2023 елның 17 гыйнварында ачык, Google Colabбушлай GPUэш вакыты.
- **Файлның зурлыгы чикләнгән дәфтәр ягы юк:** Зур аудио һәм видео файлларны эшкәртеп була.
- **Партия эшкәртү:** Берьюлы берничә аудио һәм видео файл эшкәртелергә мөмкин.
- **Google Driveярдәм:** Файлларны турыдан-туры йөкләргә мөмкин, яки Google Driveсакланган аудио / видео файлларны эшкәртеп була.
- **Онлайн:** Йөкләү һәм эшкәртү Google Colabадресы буенча башкарыла. Localирле исәпләү ресурслары кулланылмый.
- **Берничә формат:** Дәфтәр оригиналь JSONтранскрипциясен чыгара һәм TXT, CSV, Маркдаун, Excel, һәм SRTсубтитр файлларын чыгара.

Тамыры README- инглизчә керү ноктасы, һәм docs / READMEбарлыкка килгән локальләштерелгән READMEфайллары бар. Шаблон дәфтәре - чыганак дәфтәр һәм анда `{{cell1.title}}`кебек урын ияләре бар. Район файллары телгә хас READMEһәм Колабка караган күчермә бирә. Генерацияләнгән дәфтәрләр эшләнгән, шуңа күрә Колаб сылтамалары тотрыклы булып кала.


CI эш процессы шаблоннан һәм җирле файллардан READMEфайллары һәм дәфтәрләрен яңадан торгыза, дәфтәрне JSONтикшерә, буш нәтиҗәләрне тикшерә, код күзәнәкләрендә Python синтаксисын тикшерә, ачыктан-ачык очраклы серләрне сканерлый һәм барлыкка килгән файллар эшләмәгәндә уңышсыз була.

## CI нәрсә сынап карый ала

- READMEҗирле файллардан буын
- шаблоннан һәм җирле файллардан дәфтәр ясау
- дәфтәр JSONдөреслек
- буш башкару нәтиҗәләре
- Код күзәнәкләрендәге питон синтаксисы
- ачыктан-ачык очраклы серләр
- ясалган READMEһәм чыганакларга туры килгән дәфтәр файллары

Тулы транскрипция юлы Colab-специфик API-ларын, модель йөкләүләрен, GPUмөмкинлекләрен һәм кулланучылар биргән медиа-файлларны таләп итә. Бу юлны кулдан ясалган колаб төтене сынавы итеп карагыз.
