# WhisperColab

Whisperපිටපත් කිරීම සඳහා බහුභාෂා Google Colabසටහන් පොත්.

## විශේෂාංග

- **භාවිතා කිරීමට නොමිලේ:** මෙම සටහන් පොත [openai/whisper](https://github.com/openai/whisper)භාවිතා කරයි, එය 2023 ජනවාරි 17 දින OpenAIවිවෘත මූලාශ්‍රය, Google Colabහි නොමිලේ GPUධාවන කාලය සමඟ.
- **ස්ථාවර සටහන් පොත පැත්තේ ගොනු ප්‍රමාණයේ සීමාවක් නොමැත:** විශාල ශ්රව්ය සහ දෘශ්ය ගොනු සැකසිය හැක.
- **කණ්ඩායම් සැකසීම:** බහු ශ්‍රව්‍ය සහ දෘශ්‍ය ගොනු එක් ධාවනයකදී සැකසිය හැක.
- **Google Driveසහාය:** ගොනු සෘජුවම උඩුගත කළ හැක, නැතහොත් Google Driveහි ගබඩා කර ඇති ශ්‍රව්‍ය/දෘශ්‍ය ගොනු සැකසීමට හැකිය.
- **සම්පුර්ණයෙන්ම ඔන්ලයින්:** බාගත කිරීම් සහ සැකසීම Google Colabමත සිදු කෙරේ. දේශීය පරිගණක සම්පත් භාවිතා නොකෙරේ.
- **බහු ප්‍රතිදාන ආකෘති:** සටහන් පොත මුල් JSONපිටපත සහ ව්‍යුත්පන්න TXT, CSV, Markdown, Excel, සහ SRTඋපසිරැසි ගොනු ප්‍රතිදානය කරයි.
