# Whisperকোলাব

Whisperট্রান্সক্রিপশনের জন্য বহুভাষিক Google Colabনোটবুক।

## বৈশিষ্ট্য

- **ব্যবহার করার জন্য বিনামূল্যে:** এই নোটবুকে [openai/whisper](https://github.com/openai/whisper)ব্যবহার করা হয়েছে, যা OpenAIওপেন সোর্স 17 জানুয়ারী, 2023-এ Google Colab-এর বিনামূল্যে GPUরানটাইম সহ।
- **No fixed notebook-side file size limit:** Large audio and video files can be processed.
- **ব্যাচ প্রক্রিয়াকরণ:** একাধিক অডিও এবং ভিডিও ফাইল এক রানে প্রক্রিয়া করা যেতে পারে।
- **Google Driveসমর্থন:** ফাইলগুলি সরাসরি আপলোড করা যেতে পারে, অথবা Google Driveতে সংরক্ষিত অডিও/ভিডিও ফাইলগুলি প্রক্রিয়া করা যেতে পারে৷
- **সম্পূর্ণ অনলাইন:** ডাউনলোড এবং প্রক্রিয়াকরণ Google Colabএ সম্পাদিত হয়। কোন স্থানীয় কম্পিউটিং সম্পদ ব্যবহার করা হয় না.
- **একাধিক আউটপুট ফরম্যাট:** নোটবুক মূল JSONপ্রতিলিপি এবং প্রাপ্ত TXT, CSV, মার্কডাউন, Excel, এবং SRTসাবটাইটেল ফাইলগুলিকে আউটপুট করে।

## সংগ্রহস্থল বিন্যাস

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

রুট READMEহল ইংরেজি এন্ট্রি পয়েন্ট, এবং ডক্স/README-এ জেনারেট করা READMEফাইল রয়েছে। টেমপ্লেট নোটবুক হল উৎস নোটবুক এবং এতে `{{cell1.title}}`এর মতো স্থানধারক রয়েছে। লোকেল ফাইলগুলি ভাষা-নির্দিষ্ট READMEএবং Colab-মুখী অনুলিপি প্রদান করে। জেনারেট করা নোটবুক প্রতিশ্রুতিবদ্ধ যাতে Colab লিঙ্কগুলি স্থিতিশীল থাকে।

## নির্মাণ করুন

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## চেক করে

```bash
python scripts/check_notebooks.py
```

CI ওয়ার্কফ্লো টেমপ্লেট এবং লোকেল ফাইলগুলি থেকে READMEফাইল এবং নোটবুকগুলি পুনঃনির্মাণ করে, নোটবুক JSONচেক করে, খালি আউটপুটগুলি যাচাই করে, কোড সেলগুলিতে পাইথন সিনট্যাক্স চেক করে, স্পষ্ট দুর্ঘটনাজনিত গোপনীয়তার জন্য স্ক্যান করে এবং ফাইলগুলি তৈরি না হলে ব্যর্থ হয়৷

## সিআই কি পরীক্ষা করতে পারে

- READMEলোকেল ফাইল থেকে প্রজন্ম
- টেমপ্লেট এবং লোকেল ফাইল থেকে নোটবুক প্রজন্ম
- নোটবুক JSONবৈধতা
- খালি এক্সিকিউশন আউটপুট
- কোড কোষে পাইথন সিনট্যাক্স
- সুস্পষ্ট আকস্মিক গোপনীয়তা
- প্রতিশ্রুতিবদ্ধ উত্পন্ন READMEএবং নোটবুক ফাইলগুলি উত্সগুলির সাথে মেলে৷

সম্পূর্ণ ট্রান্সক্রিপশন পাথের জন্য Colab-নির্দিষ্ট API, মডেল ডাউনলোড, GPUউপলব্ধতা এবং ব্যবহারকারীর দেওয়া মিডিয়া ফাইল প্রয়োজন। জেনারেটেড-ফাইল চেক পাস করার পরে সেই পথটিকে ম্যানুয়াল Colab স্মোক টেস্ট হিসাবে বিবেচনা করুন।
