# WhisperColab

Llyfrau nodiadau amlieithog Google Colabar gyfer trawsgrifiad Whisper.

## Nodweddion

- **Am ddim i'w ddefnyddio:** Mae'r llyfr nodiadau hwn yn defnyddio [openai/whisper](https://github.com/openai/whisper), a oedd OpenAIffynhonnell agored ar Ionawr 17, 2023, ynghyd â Google Colabruntime am ddim GPU.
- **Dim terfyn maint ffeil ochr llyfr nodiadau sefydlog:** Gellir prosesu ffeiliau sain a fideo mawr.
- **Prosesu swp:** Gellir prosesu ffeiliau sain a fideo lluosog mewn un rhediad.
- **Google Drivecefnogaeth:** Gellir uwchlwytho ffeiliau'n uniongyrchol, neu gellir prosesu ffeiliau sain/fideo sydd wedi'u storio yn Google Drive.
- **Yn llawn ar-lein:** Perfformir lawrlwythiadau a phrosesu ar Google Colab. Ni ddefnyddir adnoddau cyfrifiadurol lleol.
- **Fformatau allbwn lluosog:** Mae'r llyfr nodiadau yn allbynnu'r trawsgrifiad gwreiddiol JSONac yn deillio o TXT, CSV, Markdown, Excel, a SRTffeiliau isdeitl.

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

Y gwraidd READMEyw'r pwynt mynediad Saesneg, ac mae docs/READMEyn cynnwys ffeiliau lleol READMEa gynhyrchir. Y llyfr nodiadau templed yw'r llyfr nodiadau ffynhonnell ac mae'n cynnwys dalfannau fel `{{cell1.title}}`. Mae ffeiliau Locale yn darparu copi iaith-benodol READMEa Colab. Mae llyfrau nodiadau a gynhyrchir wedi'u hymrwymo felly mae cysylltiadau Colab yn aros yn sefydlog.

## Adeiladu

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## Sieciau

```bash
python scripts/check_notebooks.py
```

Mae llif gwaith CI yn ailadeiladu READMEffeiliau a llyfrau nodiadau o'r templed a'r ffeiliau locale, yn gwirio llyfr nodiadau JSON, yn gwirio allbynnau gwag, yn gwirio cystrawen Python mewn celloedd cod, yn sganio am gyfrinachau damweiniol amlwg, ac yn methu pan nad yw'r ffeiliau a gynhyrchir wedi'u hymrwymo.

- READMEgenhedlaeth o ffeiliau locale
- cynhyrchu llyfr nodiadau o'r templed a ffeiliau locale
- llyfr nodiadau JSONdilysrwydd
- allbynnau gweithredu gwag
- Cystrawen Python mewn celloedd cod
- cyfrinachau damweiniol amlwg
- ymroddedig wedi'i gynhyrchu READMEa ffeiliau llyfr nodiadau yn cyfateb i'r ffynonellau

Mae'r llwybr trawsgrifio llawn yn gofyn am APIs penodol i Colab, lawrlwythiadau model, argaeledd GPU, a ffeiliau cyfryngau a ddarperir gan ddefnyddwyr. Triniwch y llwybr hwnnw fel prawf mwg Colab â llaw ar ôl i wiriadau ffeil a gynhyrchir basio.
