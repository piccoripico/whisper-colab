# Whisperકોલાબ

Whisperટ્રાન્સક્રિપ્શન માટે બહુભાષી Google Colabનોટબુક.

## લક્ષણો

- **વાપરવા માટે મફત:** આ નોટબુક [openai/whisper](https://github.com/openai/whisper)નો ઉપયોગ કરે છે, જે OpenAI17 જાન્યુઆરી, 2023 ના રોજ ઓપન સોર્સ, Google Colabના મફત GPUરનટાઇમ સાથે.
- **કોઈ નિશ્ચિત નોટબુક-બાજુની ફાઇલ કદ મર્યાદા નથી:** મોટી ઑડિઓ અને વિડિયો ફાઇલો પર પ્રક્રિયા કરી શકાય છે.
- **બેચ પ્રક્રિયા:** બહુવિધ ઑડિઓ અને વિડિયો ફાઇલો એક રનમાં પ્રક્રિયા કરી શકાય છે.
- **Google Driveસપોર્ટ:** ફાઇલો સીધી અપલોડ કરી શકાય છે, અથવા Google Driveમાં સંગ્રહિત ઑડિઓ/વિડિયો ફાઇલો પર પ્રક્રિયા કરી શકાય છે.
- **સંપૂર્ણપણે ઓનલાઇન:** ડાઉનલોડ અને પ્રક્રિયા Google Colabપર કરવામાં આવે છે. કોઈ સ્થાનિક કમ્પ્યુટિંગ સંસાધનોનો ઉપયોગ થતો નથી.
- **બહુવિધ આઉટપુટ ફોર્મેટ્સ:** નોટબુક મૂળ JSONટ્રાન્સક્રિપ્ટ અને વ્યુત્પન્ન TXT, CSV, માર્કડાઉન, Excel, અને SRTઉપશીર્ષક ફાઇલોને આઉટપુટ કરે છે.

## રીપોઝીટરી લેઆઉટ

```text
.github/
  workflows/
    ci.yml
docs/
  README/
    README.<language-code>.md
locales/
  *.json
notebook_template/
  Whisper_Colab.template.ipynb
notebooks/
  Whisper_Colab_<language-code>.ipynb
scripts/
  build_notebooks.py
  build_readmes.py
  check_notebooks.py
  languages.py
  repair_translated_locales.py
  sync_locales.py
  translate_full_locales.py
.gitattributes
.gitignore
LICENSE
README.md
```

રુટ READMEએ અંગ્રેજી એન્ટ્રી પોઈન્ટ છે, અને docs/READMEજનરેટ થયેલ READMEફાઈલો ધરાવે છે. ટેમ્પલેટ નોટબુક એ સ્ત્રોત નોટબુક છે અને તેમાં પ્લેસહોલ્ડર્સ છે જેમ કે `{{cell1.title}}`. લોકેલ ફાઇલો ભાષા-વિશિષ્ટ READMEઅને Colab-ફેસિંગ કૉપિ પ્રદાન કરે છે. જનરેટ કરેલી નોટબુક પ્રતિબદ્ધ છે જેથી Colab લિંક્સ સ્થિર રહે.

## બિલ્ડ

```bash
python scripts/sync_locales.py
python scripts/build_readmes.py
python scripts/build_notebooks.py
```

## ચકાસે છે

```bash
python scripts/check_notebooks.py
```

CI વર્કફ્લો READMEફાઈલો અને નોટબુકને ટેમ્પલેટ અને લોકેલ ફાઈલોમાંથી ફરીથી બનાવે છે, નોટબુક JSONચેક કરે છે, ખાલી આઉટપુટની ચકાસણી કરે છે, કોડ કોષોમાં પાયથોન સિન્ટેક્સ તપાસે છે, સ્પષ્ટ આકસ્મિક રહસ્યો માટે સ્કેન કરે છે અને જ્યારે જનરેટ કરેલી ફાઇલો પ્રતિબદ્ધ ન હોય ત્યારે નિષ્ફળ જાય છે.

## CI શું પરીક્ષણ કરી શકે છે

- READMEલોકેલ ફાઇલોમાંથી જનરેશન
- ટેમ્પલેટ અને લોકેલ ફાઇલોમાંથી નોટબુક જનરેશન
- નોટબુક JSONમાન્યતા
- ખાલી એક્ઝેક્યુશન આઉટપુટ
- કોડ કોષોમાં પાયથોન સિન્ટેક્સ
- સ્પષ્ટ આકસ્મિક રહસ્યો
- પ્રતિબદ્ધ જનરેટ READMEઅને સ્ત્રોતો સાથે મેળ ખાતી નોટબુક ફાઇલો

સંપૂર્ણ ટ્રાન્સક્રિપ્શન પાથ માટે Colab-વિશિષ્ટ API, મૉડલ ડાઉનલોડ, GPUઉપલબ્ધતા અને વપરાશકર્તા દ્વારા પ્રદાન કરવામાં આવેલી મીડિયા ફાઇલો જરૂરી છે. જનરેટ કરેલી ફાઇલ ચેક પાસ થયા પછી તે પાથને મેન્યુઅલ Colab સ્મોક ટેસ્ટ તરીકે ગણો.

## લાઇસન્સ

આ રીપોઝીટરીનું READMEજનરેટર, નોટબુક ટેમ્પલેટ, લોકેલ ફાઇલો, જનરેટ કરેલ નોટબુક્સ અને જાળવણી સ્ક્રિપ્ટો MITલાયસન્સ હેઠળ બહાર પાડવામાં આવે છે. Whisperમોડલ અને તૃતીય-પક્ષ અવલંબન તેમના સંબંધિત લાઇસન્સ દ્વારા સંચાલિત થાય છે.
